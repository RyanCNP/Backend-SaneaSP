# Backend-SaneaSP

🔗 [Repositório FrontEnd](https://github.com/MathGueff/FrontEnd-SaneaSP.git)

## Comandos:


#### Iniciando a aplicação
```
npm run start
``` 

#### Criando a documentação Swagger

```
npm run doc
```

#### Migrações

Criar

```
npm run migration-create <nome>
``` 

Fazer migrations
```
npm run migrate
```

Desfazer migrations 
```
npm run migrate:undo
```
